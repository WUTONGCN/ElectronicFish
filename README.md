# 电子木鱼 - Electron 应用

一个基于 Electron 的电子木鱼应用，支持 macOS 和 Windows 平台。

## 功能特性

- ✨ **手动/自动敲击模式**：支持手动点击或自动定时敲击
- 🎵 **音效系统**：内置两种木鱼敲击音效，可选择关闭
- 🎨 **精美视觉效果**：
  - 3D 立体木鱼设计，带渐变和阴影效果
  - 木鱼颜色自定义（支持实时渐变更新）
  - 背景颜色自定义
  - 文字颜色自定义
  - 多种木鱼样式（颜色填充/木质纹理）
  - 华丽的功德数字显示（金色渐变+发光效果）
  - 浮动文字动画（缩放+上浮效果）
- ⌨️ **快捷键支持**：自定义敲击快捷键（默认空格）
- 💾 **数据持久化**：自动保存功德计数和设置
- 🖥️ **全屏模式**：支持全屏显示
- 📱 **响应式设计**：适配不同屏幕尺寸
- 🎯 **交互反馈**：点击木鱼有缩放和阴影变化动画

## 使用说明

### 基本操作
- **敲击木鱼**：点击木鱼或按下设置的快捷键（默认空格）
- **查看功德**：左上角显示当前累计功德数
- **打开设置**：点击右上角"⚙️ 设置"按钮
- **重置**：点击"🔄 重置"按钮清空功德和设置

### 设置选项

#### 模式
- **手动敲击**：需要手动点击或按键敲击木鱼
- **自动敲击**：按设定间隔自动敲击木鱼

#### 敲击间隔
自动模式下的敲击时间间隔，单位为秒，最小值为 1 秒。

#### 音效
- **关闭**：敲击时不播放声音
- **音效1**：高音木鱼声
- **音效2**：低音木鱼声

#### 显示文字
敲击木鱼时显示的浮动文字，默认为"功德+1"。

#### 木鱼样式
- **颜色填充**：使用纯色填充木鱼
- **木质纹理1/2**：使用木质纹理（待实现）

#### 颜色设置
- **木鱼颜色**：设置木鱼的颜色
- **背景颜色**：设置应用背景颜色
- **文字颜色**：设置文字和功德计数的颜色

#### 按键设置
设置敲击木鱼的快捷键，支持：
- 数字 0-9
- 字母 a-z
- 空格（默认）

### 快捷键

- **空格**（或自定义按键）：敲击木鱼
- **Cmd+Q / Ctrl+Q**：退出应用
- **F11 / Ctrl+Cmd+F**：全屏切换
- **Cmd+R / Ctrl+R**：重新加载
